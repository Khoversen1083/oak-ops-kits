# Resources

Source-of-truth files the coach reads while running the kit. Read-only,
not student-editable. If you need to change the template, fork the
repo.

| File | What it is |
|---|---|
| `voice-rules.md` | The hard and soft voice rules the coach enforces while writing the proposal section. |
| `pricing-decision-tree.md` | The canonical decision tree: outcome value to setup fee to ongoing complexity to retainer. The IP of this kit. |
| `setup-fee-anchors.md` | What each setup fee tier ($1,000, $1,500, $2,000, $2,500) means and when to land on each. |
| `retainer-anchors.md` | What each retainer tier ($300, $400, $500, $600) covers monthly. |
| `scope-control-language.md` | Verbatim phrases for the proposal that prevent scope creep. |
| `objection-bank-pricing.md` | 10 pricing objections with a response template per objection. |

The coach reads these in this order during a run:

1. `voice-rules.md` at Step 1
2. `pricing-decision-tree.md` plus `setup-fee-anchors.md` at Step 2
3. `retainer-anchors.md` at Step 3
4. `scope-control-language.md` plus `objection-bank-pricing.md` at
   Step 4

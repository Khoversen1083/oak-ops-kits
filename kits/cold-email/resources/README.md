# Resources

Source-of-truth files the coach reads while running the kit. Read-only,
not student-editable. If you need to change the template, fork the
repo.

| File | What it is |
|---|---|
| `3-sentence-email-template.md` | The structure and example for the cold email. Subject under 8 words, three-sentence body, one ask. |
| `follow-up-sequence-template.md` | The Day 3, Day 8, Day 14 breakup sequence, all in the same thread. |
| `decision-tree-replies.md` | The 5 reply types (interested, not now, send info, remove me, hostile) with one response template each. |
| `voice-rules.md` | The hard and soft voice rules the coach enforces. Includes 5 worked bad-vs-good examples. |
| `objection-bank.md` | 12 common objections with a response template per objection. |

The coach reads these in this order during a run:

1. `voice-rules.md` at Step 1
2. `3-sentence-email-template.md` at Step 2
3. `follow-up-sequence-template.md` at Step 3
4. `decision-tree-replies.md` plus `objection-bank.md` at Step 4

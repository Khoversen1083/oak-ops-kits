# Resources

Source-of-truth files the coach reads while running the kit. Read-only, not student-editable. If you need to change a template, fork the repo.

| File | What it is |
|---|---|
| `scaling-diagnostic.md` | 20 questions, scores 8 strategic chapters 1-5. Run at Step 1. |
| `niche-validation-5-tests.md` | The 5-test gate for whether a niche scales to 10. Step 2. |
| `retainer-scaling-playbook.md` | Raise, hold, drop per client. The 30 percent minimum raise rule. Step 3. |
| `repeatable-delivery-blueprint.md` | The 80 percent reusable delivery template structure. Step 4. |
| `capacity-planning-worksheet.md` | Hours-per-client estimates and the hire-or-fire math. Step 5. |
| `first-setter-hire-guide.md` | Comp model, recruiting, week 1 spec. Step 6. |
| `cash-flow-model.md` | Real formulas for runway, MRR target, hire affordability. Step 7. |
| `pricing-power-playbook.md` | When and how to raise prices, killing low-margin clients. Reference for Steps 3 and 5. |
| `productize-decision-tree.md` | When to stop customizing each build. Reference for Step 4. |
| `12-week-scaling-plan-template.md` | Week-by-week structure with checkpoints. Step 8. |
| `voice-rules.md` | The hard and soft voice rules the coach enforces across every output. |

## The order the coach reads them

1. `voice-rules.md` once at start, hold for every output
2. `scaling-diagnostic.md` at Step 1
3. `niche-validation-5-tests.md` at Step 2 (if needed)
4. `retainer-scaling-playbook.md` plus `pricing-power-playbook.md` at Step 3 (if needed)
5. `repeatable-delivery-blueprint.md` plus `productize-decision-tree.md` at Step 4 (if needed)
6. `capacity-planning-worksheet.md` at Step 5 (always)
7. `first-setter-hire-guide.md` at Step 6 (if needed)
8. `cash-flow-model.md` at Step 7 (always)
9. `12-week-scaling-plan-template.md` at Step 8 (always)

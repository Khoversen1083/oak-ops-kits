# Sales OS Kit

The macro-kit for the operator with 1-3 paying clients who wants to scale to 5-10 without becoming an agency. Diagnoses your current state, routes you through the right tactical kits in the right order, and adds the strategic layer the tactical kits skip.

## What this kit does

You open it in Claude Code, run an interview about where the business is right now, and walk out with a 12-week scaling plan, scored against an honest diagnostic, with weekly checkpoints you can actually hit.

This is the only kit that calls into the other kits. It is the wrapper for the system.

## Who this is for

You have 1-3 paying clients. Revenue is real but irregular. You are past first-client anxiety, before agency-mode. You want a system, not more theory.

Wrong fit if:
- You have zero paying clients. Run `../cold-email/` and `../discovery-call/` first.
- You have 10+ clients and 2+ teammates. You are running an agency, this kit is too small for you.
- You want a generic business framework. This kit is opinionated. It will tell you to fire clients, hold pricing, delay hires. Different lessons need different tools.

## Before / After

| Without kit | With kit |

## Install in 3 steps

```bash
git clone https://github.com/Khoversen1083/oak-ops-kits.git
cd oak-ops-kits/kits/sales-os
claude
```

Then type: `run the kit`

## What you'll have when you're done

- A scored scaling diagnostic, 1-5 per strategic chapter
- A niche validation verdict (keep, narrow, swap) backed by the 5-test gate
- A retainer review per current client, with raise or drop calls
- An 80 percent reusable delivery template scoped to your niche
- A capacity plan with real hours-per-client estimates
- A hire-or-not decision, and if yes, a setter spec ready to post
- A 12-week cash flow forecast tied to your MRR target
- A 12-week scaling plan with weekly checkpoints

All artifacts written to `outputs/`.

## What you need

- 1-3 paying clients, with at least one on a retainer
- A locked niche (one industry, one buyer profile)
- 60-90 minutes for the first run
- Honesty about current MRR and current capacity

## How long does it take

- First full run: 60-90 minutes (Step 0 through Step 8)
- Re-run quarterly: 30 minutes, intake stays, you update numbers
- Single chapter rerun (just retainer review, just hire spec): 15 minutes each

# Objection bank

12 common objections that land in the inbox after a cold email or
follow-up. One response template per objection. Pre-load the
`<niche>`, `<pain area>`, and `<your first name>` from
`intake-filled.md` before sending.

Keep the same rules from `decision-tree-replies.md`: reply within 4
hours business hours, never pitch in the reply, one link and one ask
per send.

## 1, "We are not looking to buy anything right now"

```
Hey <firstname>,

Fair. Not asking you to buy anything, just want 15 minutes to see if
<pain area> is something worth fixing in the next quarter or two.

If the timing is not there at all, say the word, I will check back in
90 days.

<your first name>
```

## 2, "Send me more info"

```
Hey <firstname>,

Happy to. Short version, I find the one or two manual processes eating
the most time in a <niche> business and replace them with an
automation that runs in the background.

A recent build turned <process> from <time before> into <time after>.

Easiest way to know if it fits <their company> is 15 minutes. Want me
to send a booking link?

<your first name>
```

## 3, "Wrong person, talk to <other contact>"

```
Hey <firstname>,

Thanks for the redirect, much appreciated. Will reach out to <other
contact> this week.

<your first name>
```

Then write to the new contact:

```
Hey <other contact>,

<original contact> suggested you might be the right person to talk to
about <pain area>. Quick context: <one-line pitch, reference original
contact in sentence one>.

Worth 15 minutes this week?

<your first name>
```

## 4, "Interested but the timing is bad"

```
Hey <firstname>,

Got it. When does the timing usually open up for this kind of work,
end of Q? Start of next year?

Happy to put a hold on the calendar now for that window, or just check
back in a month.

<your first name>
```

## 5, "We already work with someone"

```
Hey <firstname>,

Makes sense, most <niche> owners I talk to already have someone
handling pieces of this. Where I usually fit is the specific daily
grind that does not need a full-time hire, like <pain area>.

15 minutes for an honest read on whether it is additive? If not, no
harm done.

<your first name>
```

## 6, "What's the catch / what's the price"

```
Hey <firstname>,

No catch. Most engagements look like 1K to 2.5K setup for the build,
plus 300 to 600 per month to keep it running and tweak as the
business changes.

Exact number depends on what we would actually build, which is why a
15 minute call is the best next step.

<your first name>
```

## 7, "We tried automation before and it did not work"

```
Hey <firstname>,

That is the common story, usually because the build was generic, not
built for how <niche> actually runs.

Worth 15 minutes to compare what was tried vs how I scope it? If the
shape is the same, I will tell you straight up and save us both the
time.

<your first name>
```

## 8, "Can you do <unrelated thing>"

```
Hey <firstname>,

Honest answer, not my lane. I stay tight to <pain area> and the
workflows around it.

If <unrelated thing> is the priority, I would not be the right call
for you. If <pain area> is on the list too, happy to 15 minutes on
that.

<your first name>
```

## 9, "Send me case studies / references first"

```
Hey <firstname>,

Happy to walk through a recent build live on a 15 minute call,
including what it cost, what we measured, and what would translate to
<their company>.

That is more useful than a deck. Want me to send a booking link?

<your first name>
```

## 10, "I am not the decision maker"

```
Hey <firstname>,

Got it. Who would be the right person on your side? Happy to reach
out directly and reference you if that helps.

If it is easier, I can also send a 2 line summary you can forward
internally.

<your first name>
```

## 11, "Try us again in 6 months"

```
Hey <firstname>,

Sounds good. Putting a note in my calendar for <month + 6>.

In the meantime, if anything shifts on <pain area> sooner, my email
is the same.

<your first name>
```

## 12, "Take me off your list"

```
Got it, removing you. Apologies for the noise.

<your first name>
```

Status update: Hard no, do not contact. No further sends, ever.

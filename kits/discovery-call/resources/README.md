# Resources

Source-of-truth files the coach reads while running the kit. Read-only,
not student-editable. If you need to change the template, fork the
repo.

| File | What it is |
|---|---|
| `pre-call-brief-template.md` | The 6-layer one-page brief. Company, role, signals, pains, triggers, hooks. |
| `discovery-script-template.md` | The live call script. Open, 6 questions, recap beat, close. |
| `six-question-framework.md` | The canonical 6 discovery questions and the reason each one is on the list. |
| `post-call-summary-email-template.md` | The recap email sent within 30 minutes of hang-up. |
| `voice-rules.md` | The hard and soft voice rules the coach enforces across brief, script, and email. |
| `objection-during-call-bank.md` | 6 mid-call objections with how to handle each one without breaking the discovery frame. |

The coach reads these in this order during a run:

1. `voice-rules.md` at the start, once per session
2. `pre-call-brief-template.md` at Step 1
3. `discovery-script-template.md` plus `six-question-framework.md` at Step 2
4. `objection-during-call-bank.md` is loaded at Step 2 for the user to keep open
5. `post-call-summary-email-template.md` at Step 3
